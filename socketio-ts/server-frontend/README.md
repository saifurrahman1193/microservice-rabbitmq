## Project Setup
```
npm create vite@latest
sudo chmod 777 -R *
move project folder to your root folder
npm i
npm i -D tailwindcss postcss autoprefixer
-D means development dependencies
npx tailwindcss init -p 
npx vite --port 5000 --host=0.0.0.0
```
